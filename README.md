#  Interactive Canvas Builder

A **full-stack web application** that allows users to create custom designs using **shapes, text, and images**, and export them instantly as **high-quality PDFs**.

Built with:
- **React (Vite)** on the frontend  
- **Node.js (Express) + node-canvas** on the backend

##  Features

- ** Custom Canvas Size**  
  Define specific dimensions for your workspace.

- ** Drag & Drop Interface**  
  Move rectangles, circles, text, and images anywhere on the canvas.

- ** Image Handling**  
  Add images easily via URL.

- ** Stateless PDF Export**  
  Generates a PDF of the current canvas instantly ‚Äî no database required.
